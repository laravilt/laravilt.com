# Getting Started with Laravilt

Welcome to **Laravilt** - the modern admin panel framework for Laravel + Vue.

## Introduction

Laravilt is a powerful admin panel framework that combines the best of Laravel, Vue 3, Inertia.js, and TypeScript. Build beautiful, type-safe admin interfaces with minimal effort.

## Features

- **Powerful Forms** - 30+ field types with validation and conditional logic
- **Advanced Tables** - Sorting, filtering, searching, and bulk actions
- **TypeScript First** - Full type safety from PHP to Vue
- **Vue 3 + Inertia** - Modern frontend with composition API
- **AI Integration** - Built-in AI assistants and prompts
- **Complete Auth** - Social login, 2FA, passkeys, and RBAC

## Quick Installation

```bash
composer require laravilt/laravilt
```

Then run the install command:

```bash
php artisan laravilt:install
```
